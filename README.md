# Parenting Under Pressure

The Parenting Playbook is a human-centered web project designed to support parents navigating stress, burnout, and high emotional demand. The site focuses on clarity, accessibility, and practical tools that can be used in everyday situations.

This project is being intentionally rebuilt to practice clean structure, thoughtful design, and version control as part of ongoing software development learning.

## Features
- Clear, supportive landing page
- Structured navigation across multiple pages
- Parent check-in tools and practical scenarios (in progress)
- Curated resources designed for quick reference

## Tech Used
- HTML
- CSS
- JavaScript

## Focus Areas
- Human-centered design
- Accessibility and readability
- Clear information structure
- Intentional, incremental development

## Status
This project is actively in progress and will continue to evolve as features and content are refined.
