# The Parenting Playbook
This project is a parenting support web app designed for stressed parents and caregivers who need quick, practical help in the moment. Inspired by my work teaching Nurturing Parenting concepts, the app focuses on providing simple, usable tools for emotional regulation and responsive parenting. The first version includes an emotional check-in page, where users can identify what they are feeling and receive a regulation tip, a real-life parenting scenarios page with guidance rooted in Nurturing Parenting principles, and a resource page with supportive tools and information. The goal of this project is to make parenting support feel accessible, compassionate, and immediately useful when overwhelm hits.
This project is being intentionally rebuilt to practice clean structure, thoughtful design, and version control as part of ongoing software development learning.

## Features
- Clear, supportive landing page
- Structured navigation across multiple pages
- Parent check-in tools and practical scenarios (in progress)
- Curated resources designed for quick reference

## Tech Used
